前置条件：终端下载并安装Pillow库。

pip install Pillow

Pillow是一个Python第三方图像处理库，目前是Python生态中最主流、最常用的图像处理工具之一。其核心功能包括支持图像打开、保存、缩放、裁剪、旋转、滤镜处理、格式转换、像素级操作等，常用于简单的图像批处理、生成缩略图、添加水印、图像增强等场景。

以下是对图像进行修改和格式转换的代码示例。

from PIL import Image
img = Image.open("E:/python/Example.jpg") #引号中输入图像路径
width, height = img.size
new_width, new_height = 500, 500 #修改为所需的长宽
img_resized = img.resize((new_width, new_height),Image.Resampling.LANCZOS) #按修改后的长宽缩放图片
img_resized.save("Example.webp","webp",quality=80) #修改后的名字与格式以及保存质量（保存于文件原目录下）
